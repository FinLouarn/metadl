MetaDL Challenge

Brought to you by ChaLearn and Microsoft

The competition is divided into 3 phases: Public, Development and Final. For each of these phases, the data used is different. For instance, in the Public phase, the Omniglot dataset is provided to let you familiarize with the challenge API.

Follow the instructions in the dedicated GitHub repository to get started.

Participants need to train a meta-learner on a meta-train set and produce a learner (a machine learning algorithm). This learner will be used to train on classification tasks in meta-test set and evaluated.
